# Project Cost Manager

## Descritpion

Cost Manager is going to be an application to track your costs and expenses. Make a weekly, monthly analysis of these, find your biggest expenses, and so on!

## Used technologies:

- Maven
- Spring
- Spring Data MongoDB
- Elasticsearch(in progress)
- JAX RS with Jersey

- Angular JS
- Bootstrap CSS
